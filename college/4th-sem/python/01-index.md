# Golden Viva Tips

## Rule 1

Never stay silent.

Even partial answer is better.

---

## Rule 2

Use technical keywords:

- mutable
    
- immutable
    
- interpreted
    
- dynamically typed
    
- iterable
    
- object
    

Examiners like these words.

---

## Rule 3

Give short answer first, then explain.

Example:

“Tuple is immutable collection in Python. It means its values cannot be changed after creation.”

---

## Rule 4

If you forget answer:  
Say:  
“Sir, I think…”  
Then explain logically.

Confidence matters.
