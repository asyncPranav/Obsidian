# 🔵 Lowest Priority

## Unit 3 – Inside Cloud Computing

**Probability: ⭐⭐**

This is probably the **least important unit** in your syllabus.

Reasons:

- Topics are more managerial than technical
    
- Questions are usually straightforward theory
    
- Rarely forms the entire long question in many university papers
    

Topics:

- Feeling Sensational about Organization
    
- Administering Cloud Service
    
- Managing Cloud Resources
    

These are generally easier and less frequently emphasized than virtualization, cloud services, security, etc.
