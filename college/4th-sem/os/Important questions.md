- Explain necessary conditions for deadlock (Coffman conditions).
    
- Explain Resource Allocation Graph (RAG) with example.
    

---

### 2. Deadlock Handling (VERY IMPORTANT)

- Explain Deadlock Prevention in detail.
    
- Explain Deadlock Avoidance with Banker’s Algorithm.
    
- Explain Deadlock Detection with algorithm and diagram.
    
- Explain Deadlock Recovery (process termination + resource preemption).
    

---

### 3. Banker’s Algorithm (MOST IMPORTANT NUMERICAL + THEORY)

- Explain Banker’s Algorithm with example.
    
- Solve Banker’s Algorithm problem (safe sequence question).
    
- Difference between Safe State and Unsafe State.
    
- Why Banker’s Algorithm is called avoidance algorithm?

### 4. Short but Frequently Asked (2–5 MARKS)

- What is unsafe state?
    
- Difference between prevention and avoidance.
